
# SQL

## SQL種類

- DDL(Data Definition Language)    
CREATE、DROP、ALTERなどのデータベースオブジェクト生成など    

- DML(Data Manipulation Language)    
SELECT/INSERT/UPDATE/DELETEなどのテーブルのデータ操作    

- DCL(Data Control Language)    
BEGIN、COMMIT、ROLLBACKなどのトランザクション制御
